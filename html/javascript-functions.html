A function runs a block of code. It can take arguments, or variables, and manipulate them and return them

To declare a function in JavaScript:

function functionName(argument1, argument2) { var multiply = argument1 * argument2; return multiply; }

The example above multiplies argument1 and argument2 and stores it to a new variable multiply. This multiply variable is only accessible in this function, but the function returns the value held in multiply for use from wherever you call the function. Call a function like so:

var answer = functionName(5,5);

The answer variable would contain 25.
