Cascading Style Sheets

The way to style HTML elements, and you can make them look any way you want.

To use CSS, add a link tag to the head part of your HTML document. The example below links to styles.css file in the css folder of the website directory:

In the HTML document you add the class or id attribute to the element you wish to style, or style all the HTML elements of the same name on the entire website with one command:

You can name the classes whatever you want! In styles.css you add properties to class, id using selectors and defining properties like this:

.stylishDiv { color:blue; }

In the example the . is used to denote you are selecting a class. The color property changes the color of the text inside the element.
