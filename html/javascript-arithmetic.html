A typical thing to do with numbers is arithmetic. For instance:

var x = 100 + 50; // 150

JavaScript understands most arithmetic operations, like:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus
  • ++ Increment
  • -- Decrement
You can even do arithmetic with variables:

var x = 100; var y = 25; var z = x + y; // 125
