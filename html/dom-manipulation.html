The DOM is the Document Object Model, or a fancy way of saying the website is a programming "Object" and you can enact on the model of the object using JavaScript!

If you right click on this very web page and click "View Source" this will show you the HTML source file that this website came from. Notice there is a tag abov these paragraphs that is empty!

However, as you are viewing the web page you can clearly see "DOM Manipulation" at the top. That's because I added it with JavaScript and jQuery! Just select an element using jQuery and call a method like

$('.title')text("DOM Manipulation");

and JavaScript will change the contents of the h1 tag to be the title of the web page.
